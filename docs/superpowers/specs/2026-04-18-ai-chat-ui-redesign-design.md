# AI 对话界面改版设计说明

日期：2026-04-18

## 目标

把当前“像微信的排练工具”改成“更像豆包那种 AI 助手”的中文移动端界面，同时保留原有核心功能：

- 剧本管理
- 排练模式
- 录入对话
- 图片查看
- Toast 提示

这次改版只改视觉和交互表现，不改核心业务规则。

## 已确认的设计方向

### 1. 整体气质

- 风格从“社交聊天”调整为“AI 助手对话”
- 整体更像豆包：干净、柔和、顺手、安静
- 主色使用黑、白、灰，不引入明显彩色品牌色
- 背景使用很淡的暖灰白，不做纯死白
- 整体强调留白、圆角、柔和阴影和统一图标语言

### 2. 聊天气泡与输入区

- 聊天气泡仍然左右分开，保留“我 / 对方”对话结构
- 对方消息为白色圆角卡片，带轻阴影
- 我的消息为深灰色圆角卡片，白字
- 气泡圆角、内边距、字距、图片样式统一
- 底部输入区改为豆包感的悬浮胶囊条
- 输入框默认无 placeholder 文案，保持干净
- 输入区图标改成统一的黑白灰细线风格
- 没有聊天内容时，页面不出现多余提示，也不允许整页上下滑动

### 3. 顶部、侧边栏、录入页

- 顶部弱化为轻量导航，只保留必要信息
- 剧本名作为中心标题，视觉更稳
- 左侧剧本列表更像 AI 产品里的会话面板，而不是传统菜单抽屉
- 新建、删除等动作弱化为辅助操作
- 录入页与主页面使用同一套视觉语言
- 弹窗、Toast、图片灯箱统一成同一套黑白灰风格

## 视觉原则

- 少提示，少打扰，少说明文案
- 视觉层级靠留白、圆角、阴影和字重建立，不靠大面积颜色
- 所有按钮和图标风格统一，避免拼凑感
- 保持移动端单手使用时的稳定和顺手

## 本次实现范围

- 主页面容器、背景、顶部、聊天区、底部输入区改版
- 聊天气泡、图片卡片、空状态表现改版
- 剧本侧边栏改版
- 录入页改版
- 确认弹窗、Toast、图片灯箱改版
- 图标统一成同一套风格

## 不在本次范围

- 不改排练逻辑
- 不新增语音识别或录音
- 不新增账户、云同步、分享
- 不调整剧本数据结构
